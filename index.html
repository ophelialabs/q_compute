For a computer scientist, it's not enough to write a working program, the program should be written with beautiful high level abstractions that speak to your audience. References

Genie

A full-stack web framework for Julia. Genie makes it easy to build modern web applications with Julia.

Q# IOT Project

A practical project that integrates Q# with IoT devices. Learn how to connect quantum computing concepts with real-world applications in the Internet of Things (IoT) domain.

QuTiP: Quantum Toolbox in Python

QuTiP is an open-source software for simulating the dynamics of quantum systems. It provides a comprehensive set of tools for quantum mechanics, quantum optics, and quantum information processing. QuTiP is widely used in research and education for modeling and analyzing quantum systems, making it an essential resource for anyone interested in quantum computing and quantum physics.
